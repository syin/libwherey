I built this app for fun after keeping track of a few little free library locations in a text file on my phone. See my other projects here.

tech stack and resources used

  • I forked the Countable Modern Django template repo, which sets up a boilerplate app built with Docker and Docker Compose, Django, Vue.js (which I've removed and replaced with Riot.js), and Nginx. I'm also using PostGIS instead of "regular" Postgres.
  • GIS stuff:
    • I seeded the databse with a list of libraries from the Little Free Libraries map. I don't know if this is technically allowed? But I didn't find anything against it in their usage policies, and I'm not continuously scraping their site (the JSON object is printed in console logs).
    • Leaflet.js for mapping.
    • I used Nominatim for reverse geocoding the lat/lon coordinates in a one-time batch job.
    • Finally, I used this haversine library to handle map filtering after panning or zooming. The haversine algorithm gets the distance between the corners of the visible map, which I use to get the search radius for filtering libraries.
  • The map marker icon I used is from flaticon.com.
